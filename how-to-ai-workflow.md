# How to AI Workflow

- PLAN: High-level plan (e.g. System Design)
- PLAN: Create Phases -> TODOs (the goal here is to have a phase till "MVP")
- BUILD: Test each phase after completion

- BUILD: Log all changes in a `logs/` directory for each phase or major change like a refactor
    - can use `git commits` for tracking changes with proof

or

- BUILD: Log all changes in a "general file" (use git commits) - like GitOps
    - with timestamps, problem, solution, modified files, optional notes
